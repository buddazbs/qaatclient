# IT Стартап Admin Panel

Современная админ-панель для IT стартапа с адаптивной версткой и анимациями.

## Описание

Это современная админ-панель, разработанная для IT стартапа с использованием передовых технологий и лучших практик. Приложение построено с использованием React, TypeScript и Ant Design, с архитектурой Feature-Sliced Design (FSD).

## Основные особенности

- **Современный дизайн**: Использует Ant Design для создания красивого и интуитивно понятного интерфейса
- **Адаптивная верстка**: Полностью адаптивный дизайн для всех устройств
- **Анимации**: Плавные анимации и переходы для улучшения пользовательского опыта
- **Аутентификация JWT**: Безопасная система аутентификации с использованием JWT
- **Два типа пользователей**: Администратор и обычный пользователь с разными правами доступа
- **Темная/светлая тема**: Возможность переключения между темами с сохранением выбора
- **FSD архитектура**: Четко организованная архитектура по принципам Feature-Sliced Design

## Функциональность

### Для администратора:
- Дашборд с общей статистикой
- Управление автотестами (запуск/остановка)
- Просмотр результатов тестов с возможностью перезапуска
- Отправка коллбэков
- Настройки аккаунта

### Для обычного пользователя:
- Отправка коллбэков
- Настройки аккаунта

## Технологии

- React 18
- TypeScript
- Ant Design
- Redux Toolkit
- React Router v6
- Feature-Sliced Design (FSD)

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите приложение:
```bash
npm start
```

3. Откройте [http://localhost:3000](http://localhost:3000) в браузере

## Демо учетные данные

- Администратор: `admin` / `password`
- Пользователь: `user` / `password`

## Структура проекта (FSD)

```
src/
├── app/           # Инициализация приложения и провайдеры
├── pages/         # Страницы приложения
├── widgets/       # Самостоятельные блоки функциональности
├── features/      # Бизнес-функции
├── entities/     # Бизнес-сущности
├── shared/       # Переиспользуемые модули
└── index.tsx      # Точка входа
```
